# Contributing to BeeTales Resume Builder

Thank you for helping maintain a free and private alternative for creating professional resumes.

1. Create a branch for your change.
2. Keep all personal-data processing in the browser.
3. Do not add accounts, invasive analytics, payments, or mandatory external services.
4. Preserve the separation between data, editor, and templates.
5. Add visible interface text to the centralized translation dictionaries.
6. Run `npm run build` before opening a pull request.

In the pull request, describe the problem, the solution, and the checks you performed.
